# 博弈論

## 緣起&動機

在寫程式之前，我就對於能否在電腦世界裡，快速模擬一些現實世界裡的情況，並且從中得到一些啟發感到興趣。

## 內容

在這個專案裡，我將會模擬一些簡單的博弈論情況，並且從中得到一些啟發。

## 博弈論

### 1 

想像這樣一個場景，你在上班路上打車，司機繞不繞路，你心裡沒底。你在公司談加薪，老闆會不會先壓你一手，你也沒底。但奇怪的是，在所有人都各懷心思的情況下，世界居然還能運轉得下去。市場沒天天崩，社會沒天天亂，大多數人也沒天天互坑。問題來了，如果每個人都在為自己打算盤，這個世界到底是怎麼保持秩序的？這本書《纳什均衡與博弈論》講的正是這件事。而故事要從一個我們都聽過，卻經常被誤解的人開始，亞當斯密。很多人一提到他，腦子裡立刻蹦出幾個字，看不見的手，然後順手給他扣一個帽子，他不就是那個說人都該自私點，市場自己會搞定一切的人嗎？但你如果真讀過他，就會發現，這幾乎是對亞當斯密最大的冤案之一。斯密真正想做的其實不是替自私洗白，而是想幹一件在當時非常大膽的事，給人類社會找一套像牛頓力學那樣的自然法則。牛頓告訴我們蘋果為什麼會掉下來，斯密想知道人為什麼會交易、合作、競爭，而且在沒有總指揮的情況下，還不至於亂套。注意這裡的關鍵詞不是自私，而是秩序從哪裡來。斯密非常清楚，人不是冷冰冰的理性機器，所以他在寫國復論之前先寫了一本《道德情》操論，講同情、羞愧、旁觀者的眼光、內心的道德法官。換成大白話就是，你不是只在乎錢，你還在乎別人怎麼看你、在乎自己像不像個人。你會因為不公平而憤怒，會因為佔便宜被發現而不安，會為了面子做一些並不划算的事。所以在斯密看來，所謂看不見的手，並不是在說大家放心大膽地自私吧，而是在說，在一群既自利，又有道德情緒的人之間，市場有可能形成一種穩定的協調結果。注意是，有可能，不是必然。這一點非常重要。斯密其實已經意識到一個後來反覆出現的問題。如果大家只看眼前利益，集體結果反而可能變差。只是，他那個時代還缺一套真正的數學工具，去把這種互相算計的過程說清楚。這個坑，後來被一個看起來和經濟學八竿子打不著的人填上了，逢諾依蠻。如果要用一句話形容它，那就是，人類智力上限測試器。量子力學，計算機，核武器，博弈論，他幾乎每走一步都會順手開一個全新學課。1928年，他幹了一件很奇怪的事。他沒研究市場，也沒研究企業。而是研究，賭局。兩個聰明人對賭，每一步都在猜對方下一步會怎麼出牌。不是靠運警，而是靠策略。逢諾依蠻問了一個看似簡單，但非常致命的問題。如果對方也是絕頂聰明，而且完全不犯錯，那我該怎麼做，才能保證自己最不吃虧。這就引出了一個核心思想，最小最大化原理。翻譯成人話就是不指望贏得最多，但要確保書的最少。你不是賭對方犯蠢，而是假設對方和你一樣精明。於是，一個非常反直覺的結論出現了。在很多博弈裡，最聰明的策略反而是隨機。比如石頭剪刀布，如果你有任何偏好，對方就能抓住你。唯一安全的做法就是三種出拳各佔三分之一，不是你下出，而是你在精確計算後的隨機。這件事的意義極其重大，因為它第一次把策略互動變成了一件可以嚴密計算的事情。後來，馮諾伊曼和經濟學家摩根斯特恩和寫了一本書《博弈論與經濟行為》,他們直接對傳統經濟學開炮。傳統經濟學假設什麼？假設一個人像魯賓遜一樣面對自然做選擇，而他們說，現實根本不是這樣。現實是，你的每一個選擇都在別人選擇的陰影之下。你降價是因為對手可能也會降，你不降，是因為你賭它不敢。你投資是因為別人可能跟進。你觀望是因為你怕被收割，這不是一個人的決策，而是一群人互相盯著彼此的博弈。為了把這件事講清楚，他們甚至把經濟學類比成熱力學，每個人的選擇就像分子的運動，單個分子亂跑，無法預測，但整體卻呈現穩定分佈。到這裡，私密當年平直覺看到的那種自發秩序，終於變成了一種可以被建模和證明的結構。只是，這個世界並沒有它想像的那麼溫和。因為人是策略動物，是會預判，會算計，會反擊的存在。我們被帶到了一個更極守的問題面前。如果所有人都足夠聰明，足夠理性，足夠會算計，世界真的會自動走向一個皆大歡喜的結果嗎？這個問題，之前一直沒人真正回答。直到一個性格古怪，幾乎不合群的年輕人出現。約翰納什，你可能看過《美麗心靈》,但電影其實弱化了一件事。納什最可怕的地方不是他的精神問題。而是他提出的那個思想直接改變了我們理解理性的方式。在《逢諾伊曼》的體系裡，博弈大多集中在零和博弈。你贏，我就說。你說，我就贏。可現實生活裡，大多數情況並不是這樣。比如價格戰，比如軍備競賽，比如加班，比如內卷，你多幹一點，不一定是別人少幹一點。但所有人一起多幹，可能結果反而更糟。納什問了一個非常普速的問題，在一個非零和的世界裡，理性的人最後會停在哪裡？他的答案就是後來被稱為納什均衡的東西。先別被名字嚇到。我們用一個非常生活化的例子來說，想像兩家公司都在考慮要不要降價。如果只有你降價，對方不降，你能搶走市場。如果你不降，對方降，你會輸得很慘。如果你們都降，利人一起縮水。問題是，你在決定的時候，不知道對方會怎麼選。於是，你會想，在對方不便的前提下，我有沒有理由改變自己的選擇。如果答案是沒有，那你就已經站在了一個那時均衡點上。假設現在，你已經看到對方公司在降價。這時候你只剩下兩個選項，繼續保持原價，或者跟著降價。如果你堅持不降，顧客會被對方吸走。如果你也降，至少還能保住一部分市場。於是你問自己，在對方已經降價不便的前提下，我改成不降價，會不會更好。答案是否定的？那在換一種情況，假設對方現在沒有降價，你可以選擇跟著不降，或者主動降價。如果你也不降，大家評分市場。如果你先降，你就能搶走更多客戶。於是你在問自己，在對方不降價不便的前提下，我改成降價，會不會更好。答案是肯定的，於是無論你認為對方會不會降價，你的最優選擇都會變成降價。當雙方都這樣推理，結果就是，你在降價，對方也在降價。而站在這個位置上，再回頭問那句關鍵的問題，在對方繼續降價不便的前提下，我單獨改成不降價，會不會變得更好。答案依然是否定的，於是雙方都被鎖在了這個狀態裡。這就是那時均衡，注意這個定義的殘酷之處，那時均衡並不保證這是一個好結果，它只保證一件事，在這個位置上，沒有人能單方面靠改變策略，讓自己變得更好。所以，它是一種穩定，而不是一種幸福。這就是為什麼，博弈論會顯得那麼好，因為他第一次嚴肅地告訴我們，理性不等於善良，穩定也不等於最優。求圖困境就是這個思想最經典，也是最讓人不舒服的例子。兩的嫌疑人被分開審訊，彼此無法溝通，如果兩個人都沉默，各判一年，如果一個坦白，一個沉默，坦白地立刻釋放，沉默地判十年，如果都坦白，各判五年。現在，你站在其中一個人的位置上，你並不知道，對方會選擇沉默，還是選擇坦白。於是，你開始問自己那句關鍵問題，在對方不變的前提下，我有沒有理由改變自己的選擇，先假設一種情況，假設對方選擇沉默，這時，你有兩個選擇，你也沉默，判一年，或者你坦白，立刻釋放，於是你會問，在對方沉默不變的前提下，我改成坦白，會不會更好？答案是會，那在換一種情況，假設對方選擇坦白，這時，你也有兩個選擇，你沉默判十年，或者你也坦白，判五年，於是你在問，在對方坦白不變的前提下，我改成坦白，會不會更好？答案依然是會，於是你發現一件事，無論你認為對方會怎麼選，對你來說，坦白都是更有力的選擇。當兩個人都這樣推理，結果就是，你坦白，對方也坦白，但結局卻是，兩個人都比合作沉默判的更重，這不是人性壞，而是邏輯本身，把你推到了這個位者。這也是納什均衡最震撼人的地方，他揭示了一個長期被忽略的事時，個體理性並不自動生成集體理性。再換一個現代一點的現想。所有人都不加班，效率也許沒那麼高，但生活還過得去，一旦有人開始加班，其他人就被迫跟上，最後大家都再加班，但晉升名額並沒變多。這是不是一種穩定狀態？是的，是不是一種好的狀態？未必，但你身處其中，卻很難單方面退出，這正是納什均衡的牢龍感，你不是不知到更好的世界存在？你只是到不了，很多人聽到這裡會下意識反駁，那大家商量一下不就好了。問題是，博弈論恰恰研究的就是在無法完全信任，無法強制執行承諾的情況下，世界會走向哪理？這也是為什麼？納什均衡很快就從經濟學擴散到了政治學國際關係，甚至生物學。軍備競賽為什麼停不下來？不是因為誰特別好戰？而是因為在對方不裁軍的情況下，你裁軍反而更危險。這不是到的問題，而是結構問題。但故事並沒有到此為止。如果世界真的只能停在這些尷尬的穩定點，那人類社會早就玩完了。可現實是，我們確實看到了大量合作，陌生人之間的合作，沒有警察盯著的合作，甚至帶有犧牲意味的合作。問題來了，博弈論是不是低估了合作？答案，來自一個名字看起來毫不起眼的人，約漢梅納德史密斯。注意，不是亞當斯密是進化生物學家史密斯。他把博弈論直接丟進了自然選則理。他的思路非常簡單也非常殘忍。動物不會算數學，那他們的策略是怎麼來得？答案是活下來的就是策略正確的。史密斯提出了一個概念，進化穩定策略。意思是，一旦某種行為策略在群體中佔了上風，任何變異策略都很難入清成功。這聽起來是不是很像那時君衡？沒錯，那時君衡是理性穩定。進化穩定策略是進化穩定。一個靠思考，一個靠淘汰。史密斯用一個非常生動的例子解示這一點。鷹鴿博弈。想向動物爭奪食物時有兩種行為。鷹策略一上來就打，拼命。鴿子策略不打駕，只展示嚇唬。情況會使。鷹遇到鴿子鷹贏，吃到食物。鴿子遇到鴿子平分。鷹遇到鷹雙方受傷，代價很大。現在問，如果全世界都是鴿子突然出現一隻鷹，它會不會混得很好。會因為它每次都欺負鴿子。所以全世界鴿子不穩定。那如果全世界都是鷹呢？大家天天硬剛天天受傷死亡率很高。這時如果出現一隻鴿子，它專門躲開衝突反而可能活得更久。所以全世界鷹也不穩定。結果是什麼？進化出來的通常是一部分鷹一部分鴿子。這個比例一旦形成。不管你多投放一點鷹還是多投放一點鴿子。它們都很難改變整體結構。這個比例就是進化穩定策略。這一件是意義非常大。因為它告訴我們合作退讓甚至看起來吃虧的行為並不一定是反理性的。它們可能是在更大之度上穩定的。它們從人類算計延生到生命演化 是凱特勒提出了一個概念平軍人。意思不是說真有這麼一個人存在。而是說當你把足夠多的人放在一起，個體的任性會被抵消整體反而呈現規律。這在當時簡直是對自由意治的正面挑性。但這個思想很快被另一個領域揭走了。物理學麥克斯偉他研究的是氣體分子。當個分子怎麼運動？你根本預測不了。他亂非亂狀亂談。但如果你有上一個分子你反而能寫出幾期精確的功勢。溫度壓力體機全都穩定可算。麥克斯偉意識到一件是統計規律不需要個體是理性的。他只需要數量足夠大。這件是對博義論意為著什麼？意為著一個大膽的可能性。也許我們根本不需要知道每個人的心理動機情續。只要知道他們如何連接整體行為就能被理解。這正式網落科學登場的地方。故時的入口其實很輕鬆。來自一個娛樂遊戲叫凱文培根遊戲。規則很簡單。給你一個演員的名字讓你不斷去走。這個演員和誰一起演過電影。那個人又和誰一起演過。一層一層往下連。看最少需要幾步才能連到演員凱文培根。一開始大家以為這只是的玩笑。但很快發現一件很有意思的是，大多數演員真的只需要六部以內就能和凱文培根建理聯系。於是這個遊戲變成了一種可以測良的網落特形。他在告訴我們社惠網落既不是完全隨機的也不是嚴格規則排裂的。而是處在一個非常微妙的中間狀態。後來瓦詞和斯託家詞用一個極期簡單的模形證名。只要在原本狠規則的網落中隨機加上少量遠程連接，整個網落的平軍路境長度就會突然大幅下降。你會發現好像任何人都能在很短的距離內被找到。這就是所為的小事界。你可以把他想像成一個村裝。房子排成一排每一戶人家只認識左右兩邊的鄰居。在這種情況下如果你住在最左邊想聯系到最右邊的人，只能一家一家的船過去要走很久。但現在假社村字裡多了幾條奇怪的熟人關係。比如最左邊的人認識了中間的一戶，中間某戶又認識了最右邊的人。你會發現原本要走七八步的路，現在可能只需要兩三步。結構幾乎沒變，只是多了幾條捷境，整個世界卻突然縮小了。你平時認識同事認識同學認識鄰居。這是本地連接。但你某個同學去了國外，你某個親奇在大公私，你網友在另一個城神，這些就是捷境，只要存在少量這種關係，整個社會就會變成小世界，而更驚人的發現還在後面，巴拉巴西等人發現，真時世界裡的網路根本不是大家差不多，而是即少數捷點擁有海量連接，絕大多數捷點只有很少連接，也就是說世界不是平軍分配的，而是頭部戰劇大部份資源，比如在互聯網世界，大多數網戰每天只有幾百個訪問，而少數幾個網戰每天卻有上一訪問，再比如社交平臺，大多數人只有幾十個粉私，少數網紅卻擁有幾百萬上千萬粉私，再比如城事，大多數城事規模普通，但少數幾個超級城事，聚集了全國最密級的人口資本和機會，這類結構叫做無標度網臥，而且一代你成為頭部，你會更容易繼續獲得連接，從而繼續保持頭部，新節點進來時，更容易連接有名的，新用戶建網戰，更可能連接到谷歌，而不是一個沒人知道的小博課，新演員，更想和大明星合作，而不是無名群演，於是有名的人越來越有名，所以頭部很難被撼動，這現象和博議論突然產生了化學反應，因為一代你承認，人與人之間的博議不是在真空裡發生，而是在關係網落中發生，那很多就結論就站不住了，在一個完全隨機的社會裡，合作很難活下來，比如你今天跟誰交易，明天跟誰打交到，基本全靠抽遷，你坑為一個人，大概率這輩子也不會再見到，那理性選擇是什麼，先站便易再說，所以傳統模行裡自思是優視策略，但現時世界不是這樣，現時更像什麼，你每天見到的是同一批人，同事鄰居同學熟人朋友圈，於是博義發生在關係網落裡，而不是發生在全社會大抽獎，這時候情況就完全變了，舉個簡單的例子，一個小區裡有五家經常互相來往，今天你幫我收個快地，明天我幫你招個孩子，後天一起拼個團狗，大家都合作日子很好過，突然有個人騙了一次錢，結果會怎樣，消息很快傳開，群裡說，鄰居一寮，這個人以後沒人理她了，他也許賺到一次，但徹底被排出在圈子外，於是你會發現，在這個小圈子裡，合作法是最劃算的，再往下推一步，慢慢地，願意講信用的人，會更願意跟講信用的人來往，愛佔便義的人，只能和愛佔便義的人混在一起，於是社會開始分層，好人找好人，壞人擠壞人，不是誰設計的，而是結構自然形成的，這也讓三件事突然變得極其重要，生慾，重復互動，鄰裡關血，因為你做的每一件事，都會在社會網路裡留下痕記，別人會記主，會傳播，會形成標簽，久而久知，你能進入什麼圈子，你能和誰合作，其實早就被決定了，這也解示一個很現實的問題，為什麼有些社區特別有人情味，而有些地方永遠冷沒，不是因為這裡的人天生更善良，也不是因為那裡的人人品更差，而是網路結構不同，熟人多，圈子小往來密急，信人就容易生長，人流快莫生人多，關係一次信，信人就很難見理，所以結論其實很殘酷，很多時候不是人選擇了自私，而是所處的網路講力自私，也不是人突然變高賞，而是所處的網路講力合作，當博議論網路科學統機物理開始融合一個新的詞出現了，社會物理學，他不是說社會真得像物理樣簡單，而是說我們可以用理解複雜系統的方式去理解社會，很多社會現向其實不是慢慢先行變化的，而是前面看起來風停浪靜突然一下子整體翻轉，比如輿論一開始只是零行幾個人在吐巢，後來吐巢的人慢慢變多，某一天開始你會發現幾乎所有人都在罵，沒有人下命領沒有人統意安排，但社會態度突然換了一種方下，再比如集體行動本來大家各自猶欲，要標辭職，要標跳走，要標搬走，慢慢的身邊開始有人動了，接著你發現一批人在差不多的時間做出了差不多的選擇，好像情虛會傳染猶欲會共振，還有市場崩潰，新聞總喜歡說，因為某條消息市場暴跌，但真正的情況往往逝，債務早就很高，泡沫早就很大，風險堆了很多年，那條消息只是最後一根稻草，重要的不是哪根稻草壓斷了落脫，而是落脫早就快被壓垮了，這些現象背後有一個共同點，系統在某個階段已經站在了危險邊源，只差一點點就會整體翻車，這時候博議論的角色也發生了變化，他不在指事，研究兩個人怎麼對抗，三個人怎麼算及，而是變成了，每個人都在按自己的利義做選責，這些選責通過人與人之間的關係網落，不斷傳地，放大，疊家，最後在宏官層面形成某種集體結果，換句話說，沒有誰掌控全局，但結果看起來像是有規律的，每個人只是在過自己的日子，但整個社會卻呈現出某種可以被研究的模式，這也讓一個原本聽起來很狂望的想法，變得越來越現實，如果我們真的掌臥了，這些規律是不是就能預測社會，不是預測某一個人，而是預測趨勢風險拐點，這個念頭並不是今天才出現的，早在幾十年前科患小說裡就已經有人大膽寫過這種社想，一個名字開始頻繁被提起，阿西莫夫，而他虛購的那門學科也開始從幻想，慢慢走向現實門口，回頭看一下剛才走過的路，一開始我們把人當成理性的計算者，後來發現人會被情序影想，會憤怒會懲罰會衝動，再後來我們乾翠不再盯著單個人，而是把人放進網落結構理，放進統計規律理，於是一個問題已經無法迴避，如果我們不預策個人，只預策整體，社會是不是也能被算出來，早在幾十年前，阿西莫夫在他的基地系列裡，有一門虛購的學科叫心理史學，核心思想只有一句話，無法預策個人，但可以預策億萬人組成的社會，聽起來是不是很熟？你會發現，我們剛剛講過的統計物理、網絡科學、博弈論，幾乎都在朝這個方向前進，區別只在於，阿莫希夫是用小說寫的，而現實世界開始真的有人嘗試用公式做這件事：社會物理學就是這個嘗試的現實版本，研究者把輿論從眾極化；合作、衝突、用賴四物理模型的方式去描述，比如伊辛模型，原本是研究磁性材料的，後來被拿來解釋為什麼人群會突然形成一致觀點，或者突然對歷程兩派，再比如輿論傳播模型，解釋謠言如何擴散，解釋為什麼沉默的大多數會在某格時刻突然集體發生，這些模型都非常簡化，甚至有點粗暴，但他們有一個共同點，不關心你是誰，只關心你和誰相連，以及你如何被影響，這正是心理史學的核心精神，當然，這條路爭議極大，因為人不是分子，你有技藝，信仰、叛逆辛、任何過度簡化都有可能變成胡扯，但作者在這裡強調了一件非常重要的事情，模型的價值，不存在於完全正確，而在於是否有洞察力，就像氣象模型，他永遠算不准，明天下午三點會不會下雨，但他能告訴你颱風會不會來，社會模型也是一樣，他們不預測你會怎麼做，而是預測，在什麼條件下，系統會變得不穩定，而就在這個問題上，博弈論迎來了一個看起來最離譜，但邏輯上卻非常嚴肅的分支：量子博弈論，故事的切入點，居然是一枚硬幣，想像一個簡單的遊戲，桌上有一枚硬幣，正面朝上，兩個人輪流操作，每人一次，規則只有一個，你可以選擇翻或者不翻，最後如果硬幣是正面，A迎，如果是反面 B迎。在經典規則下，這是一個公平到無聊的遊戲，雙方只要隨機操作，勝率各一半，但大衛梅耶,在1998年提出了一個問題，如果其中一方能使用量子策略呢？所謂量子策略，並不是算得更快，也不是堵得更很，而是你連正面還是反面這個前提都可以先放一邊，在量子世界裡，硬幣在被真正看見之前，可以處在一種疊加態，量子玩家在做的事，不是選擇要不要翻面，而是先把硬幣送進這種狀態，等對方操作完，再把結果鎖定成對自己有利的那一面。
